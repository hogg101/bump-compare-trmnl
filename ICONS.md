- **Format**: SVG (Scalable Vector Graphics)
- **Size**: Vector-based, scalable to any size
- **Style**: Bold silhouettes, high contrast, suitable for e-ink displays
- **Background**: Transparent
- **Naming**: Must match filename exactly (case-sensitive)
- **Design**: Monochrome/1-bit style, simple shapes optimized for e-ink rendering
